Blender Prototype Project

For the final version of my Blender prototype, I utilized the feedback that I received in class to create a more dynamic design. To do so, I created a completely new design that allowed for movement and on-site durability. In terms of inspiration, I considered the Pipboy, Kindle, and Nintendo Switch.

I relied on many tutorials throughout this process. I learned a lot during this project and felt like it gave me a better grasp of Blender. Two tools that really helped me move along were using keyboard shortcuts with X/Y/Z, and BlenderKit. There are some elements that I would like to keep working on as I make this into a portfolio piece, but overall, I think it effectively displays my GDACS environmental dashboard.
